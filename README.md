# 🏭 Система учёта смен и производства

> Комплексное решение для управления рабочими сменами, производством и складами на пластиковом производстве

## 📋 Описание

Desktop и веб-приложение для автоматизации учёта рабочих смен, производства продукции, управления складами сырья и заготовок, а также расчёта заработной платы сотрудников.

## ✨ Возможности

### 👥 Управление пользователями
- Три роли: Сотрудник, Руководитель, Администратор
- Регистрация и авторизация
- Сброс паролей администратором

### ⏰ Учёт смен
- Открытие/закрытие смены с таймером
- Учёт опозданий
- История смен с детализацией
- Мониторинг сотрудников в реальном времени

### 🏭 Производство
- Учёт взятых заготовок
- Фиксация выпуска годной продукции и брака
- Автоматическое списание заготовок со склада
- Учёт недостачи с указанием причины

### 📦 Склады
- Управление складом сырья (приход/расход)
- Управление складом заготовок
- Отслеживание остатков в реальном времени

### 💰 Финансы и ЗП
- Настройка систем оплаты (почасовая, сдельная, оклад)
- Расчёт заработной платы за период
- Экспорт отчётов в Excel

## 🛠️ Технологии

- **Backend:** FastAPI (Python 3.13)
- **Desktop:** PySide6 (Qt6)
- **База данных:** SQLite (Async SQLAlchemy)
- **Веб:** HTML/CSS/JS (PWA)
- **Аутентификация:** JWT (python-jose)
