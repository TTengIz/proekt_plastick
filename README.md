# 🏭 Учёт смен и производства

Система учёта производственных процессов, управления складом и мониторинга сотрудников.

## 📋 Описание

Десктопное приложение для автоматизации учёта на производстве:
- Учёт рабочих смен
- Управление складом сырья и заготовок
- Контроль производства и брака
- Мониторинг сотрудников
- Расчёт зарплаты
- График отпусков
- Технологические карты (рецептуры)

## 🚀 Возможности

### ✅ Учёт смен
- Начало и завершение смены
- История смен
- Подсчёт отработанных часов

### 🏭 Производство
- Списание заготовок
- Учёт годных изделий и брака
- Автоматическое списание сырья по рецептурам
- История производства

### 📦 Склад сырья
- Приход и расход материалов
- Учёт по толщине и цвету
- Редактирование остатков
- Автоматическое списание при производстве

### 📋 Склад заготовок
- Учёт готовых изделий
- Быстрое добавление/изменение
- Автоматический расчёт расхода материалов

### 👥 Мониторинг сотрудников
- Статусы: работает, опоздал, не вышел, больничный, отпуск
- Цветовая индикация
- Отслеживание опозданий

### 💰 Финансы и зарплата
- Сдельная и почасовая оплата
- Расчёт зарплаты за период
- Экспорт отчётов

### 📅 График отпусков
- Планирование отпусков
- Учёт больничных
- Доступ для руководителей

### 📐 Технологические карты
- Настройка рецептур
- Автоматический расчёт расхода сырья
- Учёт изделий в листе материала

## 🛠 Технологии

- **Backend**: Python + FastAPI
- **Frontend**: Python + PySide6 (Qt)
- **Database**: PostgreSQL
- **ORM**: SQLAlchemy (async)
- **Auth**: JWT tokens
